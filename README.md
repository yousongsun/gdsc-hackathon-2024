# HackBar Team - GDSC Hackathon 2024 Repository

## Run the App

### backend

In the backend folder

- set the .env file
- install the dependencies
  ```
  npm i
  ```
- run the backend
  ```
  npm start
  ```

### frontend

In the frontend folder

- install the dependencies
  ```
  npm i
  ```
- run the frontend
  ```
  npm run dev
  ```
